google浏览器右键功能菜单扩展
=================


在google中选择文本之后，右键可以再不同的搜索引擎中搜索相关的信息

资源
----------------
官网地址：无

源代码及最新更新：https://github.com/Dn9x/RightMenu


使用方法：
--------------
1.在google浏览器中：设置->扩展程序->载入正在开发的扩展程序(要勾选开发者模式)
